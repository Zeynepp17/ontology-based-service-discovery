# Ablation Study

## Ontology Distance Effect

Comparing Ontology V1 and Ontology V2 shows that ontology-distance based similarity significantly improves retrieval performance.

F1 Score:

- Ontology V1: 0.4603
- Ontology V2: 0.5776

This demonstrates that ontology depth information contributes substantially to semantic service discovery.

---

## Hybrid Retrieval Effect

Comparing Ontology V2 and Hybrid Matching shows that combining lexical similarity with ontology-based similarity further improves retrieval quality.

F1 Score:

- Ontology V2: 0.5776
- Hybrid Matching: 0.6192

The hybrid approach achieved the highest overall performance across all evaluation metrics.
