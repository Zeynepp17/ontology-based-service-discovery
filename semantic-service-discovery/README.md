# Ontology-Based Service Discovery

This project focuses on ontology-based semantic service discovery in the travel domain.

## Project Scope

The project aims to compare:

- Keyword-based service discovery
- Ontology-based semantic matching
- Hybrid semantic retrieval approaches

## Dataset

### Service Dataset
- 60 travel domain services
- OWL-S inspired semantic structure
- Inputs / outputs / preconditions / effects
- QoS-aware service representation

### Query Dataset
- 40 semantic user queries
- Exact match labels
- Plugin match labels
- Partial match labels
- Irrelevant labels

## Domains

- Accommodation Services
- Transportation Services
- Tourism Services
- Planning Services
- Documentation Services
- Insurance and Payment Services
- Safety and Weather Services

## Planned Evaluation Metrics

- Precision@k
- Recall@k
- F1-score
- nDCG
- Query execution time

## Technologies

- Python
- JSON
- TF-IDF
- Ontology-based semantic matching
